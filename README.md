# SMART_LAMP
스마트램프(IoT구현)-블루투스 스마트어플로 제어(어플제작)
- 블루투스 모듈을 활용하여 LED색깔을 내맘대로 조정하는 스마트폰 어플제작
- 네오픽셀 라이버러리
- DHT11,22(온습도센서)
- 블루투스(HC-06)
- I2C LCD 라이버러리
